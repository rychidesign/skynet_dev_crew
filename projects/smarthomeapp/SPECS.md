# SmartHome App

## Popis
Desktopová aplikace pro ovládání chytrého domu.

## Tech Stack
- Framework: Electron.js
- Frontend: React + TypeScript
- UI: Tailwind CSS

## Cíl
Aplikace pro Windows/Mac pro monitoring a ovládání IoT zařízení.

## Funkce
- Dashboard s přehledem zařízení
- Ovládání světel, termostatu, zámků
- Notifikace
- Historie událostí
